This course covers fundamental programming concepts that form the foundation for effective and high-quality learning in professional modules on automated testing.
The course covers:
-Writing simple program code (basic coding skills)
-Working with an Integrated Development Environment (IDE)
-Using variables and data types
-Applying operators and expressions
-Reading input and printing output to the console
-Utilizing conditional statements (if, if-else) and loops (for, while, do-while)
-Working with arrays
-Working with lists
-Performing component-level testing using NUnit
